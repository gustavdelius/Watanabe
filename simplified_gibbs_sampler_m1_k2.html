Gibbs Sampler for Normal Mixture (Simplified for \(M=1, K=2\))

7.2.1 Gibbs Sampler for Normal Mixture

Gibbs sampler is often employed in mixture models. Let us derive an algorithm by Gibbs sampler for a normal mixture. In this document, a normal distribution of \(x \in \mathbb{R}\) for a given mean \(b \in \mathbb{R}\) is denoted by \[ \mathcal{N}(x|b) = \frac{1}{\sqrt{2\pi}} \exp\left(-\frac{(x - b)^2}{2}\right) \]

Then a normal mixture with two components (\(K=2\)) is defined by \[ p(x|a, b) = a \mathcal{N}(x|b_1) + (1-a) \mathcal{N}(x|b_2) \] where \(a \in [0, 1]\) and \(b = (b_1, b_2)\) are parameters of the normal mixture. For the prior, we adopt \[ \varphi(a) = \frac{1}{z_1} a^{\alpha - 1} (1-a)^{\beta - 1} \] \[ \varphi(b) = \frac{1}{z_2} \exp\left(-\frac{b_1^2 + b_2^2}{2\sigma^2}\right) \] where \(\varphi(a)\) and \(\varphi(b)\) are the Beta distribution with index \((\alpha, \beta)\) and the normal distribution, respectively. Here \(\alpha, \beta > 0\) and \(\sigma^2 > 0\) are hyperparameters and \(z_1, z_2 > 0\) are constants. Let \(y \in \{0, 1\}\) be a latent variable indicating the component membership.

Then a statistical model for the simultaneous probability density of \((x, y)\) is defined by \[ p(x, y|a, b) = (a \mathcal{N}(x|b_1))^{y} ((1-a) \mathcal{N}(x|b_2))^{1-y} \] It follows that \[ p(x|a, b) = \sum_{y \in \{0, 1\}} p(x, y|a, b) \] Therefore a normal mixture \(p(x|a, b)\) can be understood as a statistical model \(p(x, y|a, b)\) which has a latent or hidden variable \(y \in \{0, 1\}\).

Let \(x^n = \{x_1, x_2, \dots, x_n\}\) be an independent sample and \(y_i \in \{0, 1\}\) be the hidden variable which corresponds to a sample point \(x_i\). We use a notation \(y^n = \{y_1, y_2, \dots, y_n\}\). Then Bayesian simultaneous probability is \[ p(a, b, x^n, y^n) = \frac{1}{Z} \left(a^{\alpha - 1 + n_1} \exp(-H_1(b_1))\right) \left((1-a)^{\beta - 1 + n_2} \exp(-H_2(b_2))\right) \] where \(Z\) is a normalizing constant and \[ n_1 = \sum_{i=1}^n y_i, \quad n_2 = \sum_{i=1}^n (1 - y_i) \] \[ H_1(b_1) = \frac{b_1^2}{2\sigma^2} + \frac{1}{2} \sum_{i=1}^n y_i (x_i - b_1)^2 \] \[ H_2(b_2) = \frac{b_2^2}{2\sigma^2} + \frac{1}{2} \sum_{i=1}^n (1-y_i) (x_i - b_2)^2 \]

In Bayesian estimation, we need the posterior parameters \((a, b)\) which are subject to \(p(a, b|x^n)\). If \((a, b, y^n)\) are subject to \(p(a, b, y^n|x^n)\), then \((a, b)\) of them can be used for numerical approximation of the posterior distribution. Therefore it is sufficient to make a Gibbs sampler for \((a, b)\) and \(y^n\), in which the conditional probabilities we need are \(p(y^n|a, b, x^n)\) and \(p(a, b|x^n, y^n)\).

By using these conditional probabilities, we can make a Gibbs sampler for \((a, b) \to y^n\) and \(y^n \to (a, b)\). Under the probability distribution \(p(y^n|a, b, x^n)\), \(y_1, y_2, \dots, y_n\) are independent and, for each \(i\): \[ p(y_i = 1 | a, b, x_i) = \frac{a \mathcal{N}(x_i|b_1)}{a \mathcal{N}(x_i|b_1) + (1-a) \mathcal{N}(x_i|b_2)} \]

On the other hand, if \((a, b_1, b_2)\) is subject to the probability distribution \(p(a, b|x^n, y^n)\), they are independent. Hence \[ p(a, b|x^n, y^n) = p(a|x^n, y^n) p(b_1|x^n, y^n) p(b_2|x^n, y^n) \] The variable \(a\) is subject to the Beta distribution with index \(\alpha + n_1\) and \(\beta + n_2\): \[ p(a|x^n, y^n) \propto a^{\alpha + n_1 - 1} (1-a)^{\beta + n_2 - 1} \]

The variable \(b_1\) and \(b_2\) are subject to the normal distribution with average \(b_k^*\) and variance \((\sigma_k^*)^2\): \[ p(b_k|x^n, y^n) = \mathcal{N}(b_k^*, (\sigma_k^*)^2) \quad \text{for } k=1, 2 \] where \[ b_1^* = \frac{\sum_{i=1}^n y_i x_i}{1/\sigma^2 + n_1}, \quad b_2^* = \frac{\sum_{i=1}^n (1-y_i) x_i}{1/\sigma^2 + n_2} \] \[ (\sigma_1^*)^2 = \frac{1}{1/\sigma^2 + n_1}, \quad (\sigma_2^*)^2 = \frac{1}{1/\sigma^2 + n_2} \]

Hence we obtained the Gibbs sampler.

Gibbs Sampler for Normal Mixture (\(M=1, K=2\))

  1. A parameter set \(\{(a, b_1, b_2)\}\) is initialized.
  2. A set of hidden variables \(\{y_i\} \in \{0, 1\}^n\) is determined by the probability: \[ P(y_i = 1|a, b, x_i) = \frac{a \mathcal{N}(x_i|b_1)}{a \mathcal{N}(x_i|b_1) + (1-a) \mathcal{N}(x_i|b_2)} \]
  3. Using \(n_1 = \sum_{i=1}^n y_i\) and \(n_2 = n - n_1\), a parameter \(a\) is generated by using the Beta distribution: \[ a \sim \text{Beta}(\alpha + n_1, \beta + n_2) \]
  4. A parameter \(\{b_1, b_2\}\) is generated by the normal distribution \(\mathcal{N}(b_k^*, (\sigma_k^*)^2)\) for \(k=1, 2\) where \[ b_1^* = \frac{\sum_{i=1}^n y_i x_i}{1/\sigma^2 + n_1}, \quad (\sigma_1^*)^2 = \frac{1}{1/\sigma^2 + n_1} \] \[ b_2^* = \frac{\sum_{i=1}^n (1-y_i) x_i}{1/\sigma^2 + n_2}, \quad (\sigma_2^*)^2 = \frac{1}{1/\sigma^2 + n_2} \]
  5. Return to (2).
